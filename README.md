# Java Coursework Portfolio

Selected Java projects demonstrating my progress in GUI development, object-oriented programming, database connectivity, and secure coding practices.

## Projects

### [1. Student Grade Management System](student-grade-system/)

A Java Swing desktop application for entering and displaying student records.

![Student Grade Management System](https://github.com/user-attachments/assets/c5fc27a8-10c2-45fe-92c1-8587a13e0277)

Features:

- Enter student ID, name, and score
- Display submitted student records
- Calculate the average score
- Clear and reset form inputs
- Validate missing and invalid input

Concepts:

- Java Swing
- Event handling
- GUI layout management
- Classes and objects
- `ArrayList`
- Exception handling

### [2. JDBC and MySQL Grade Database](jdbc-mysql-practice/)

A Java console application demonstrating secure database access and SQL operations.

Features:

- Create a grade table
- Insert and update student records
- Query and sort course grades
- Delete records based on conditions
- Read credentials from environment variables

Concepts:

- JDBC and MySQL
- SQL CRUD operations
- `PreparedStatement`
- `ResultSet`
- Try-with-resources
- Secure credential handling
