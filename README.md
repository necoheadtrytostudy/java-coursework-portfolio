# Java Coursework Portfolio

A collection of selected Java coursework projects demonstrating my progress in GUI development, object-oriented programming, database connectivity, file I/O, and multithreading.

## Projects

### 1. Student Grade Management System

A Java Swing application for entering and displaying student records.

Features:

- Enter student ID, name, and score
- Display submitted student records
- Calculate the average score
- Clear and reset form inputs
- Handle button events through `ActionListener`

Concepts:

- Java Swing
- Event handling
- GUI layout management
- Arrays and data processing

### 2. JDBC and MySQL Practice

Exercises connecting Java applications to a MySQL database.

Features:

- Insert student grade records
- Update existing records
- Query and sort course grades
- Delete records based on conditions
- Use `PreparedStatement` for parameterized input

Concepts:

- JDBC
- MySQL
- SQL CRUD operations
- Database connections

### 3. Object-Oriented Library System

A console-based inventory exercise for textbooks and novels.

Concepts:

- Classes and objects
- Inheritance
- Abstract classes
- Method overriding
- Arrays and inventory management

## Additional Practice

- File input and output
- Basic multithreading
- Java Swing interface exercises
- Exception handling
