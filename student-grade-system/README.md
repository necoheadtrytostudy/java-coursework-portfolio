# Student Grade Management System

A desktop application developed while learning Java Swing and object-oriented programming.

## Features

- Add a student ID, name, and score
- Display saved student records
- Calculate the average score
- Clear input fields
- Validate missing, non-numeric, and out-of-range scores

## Concepts Practiced

- Java Swing
- Event handling with `ActionListener`
- GUI layouts
- Classes and objects
- `ArrayList`
- Exception handling
- Input validation

## Run the Application

Requirements: JDK 8 or newer.

```bash
cd src
javac StudentGradeSystem.java
java StudentGradeSystem
